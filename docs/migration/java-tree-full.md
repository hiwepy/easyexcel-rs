## 5. Source-of-truth convention

- All Rust ports must reference the original Java file path via doc comments
  (e.g. `/// Mirrors Java: com.alibaba.excel.EasyExcelFactory.write(File, Class)`)
- The Rust function signature's first parameter type and last parameter type must
  match the Java method signature (modulo language-idiomatic adaptation).
- Public Rust trait methods must expose all lifecycle hooks defined in the Java
  handler hierarchy (before/after × workbook/sheet/row/cell × create/dispose/data-converted).
