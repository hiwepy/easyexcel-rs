## 12. Consistency rules for new code

```rust
// Every Rust file porting Java code MUST start with the following:
/// Mirrors Java: com.alibaba.excel.<full.path>.<ClassName>
/// Java class: <copy original Java class docstring>
//
/// ## Methods ported
/// - `method_name(ParamType) -> ReturnType` -> `method_name(&self, param: &ParamType) -> Result<ReturnType>`
/// - ...
```

For Rust derive-macro attributes, the corresponding Java annotation MUST be referenced:

```rust
#[derive(ExcelRow)]
/// Mirrors Java: com.alibaba.excel.test.core.simple.SimpleData
struct SimpleData {
    /// Mirrors Java: @ExcelProperty("姓名")
    #[excel(name = "姓名")]
    name: String,

    /// Mirrors Java: @ExcelProperty(index = 1)
    #[excel(index = 1)]
    age: u32,
}
```
